# Smart-Multi-Campus-ICT-Infrastructure
An SD-WAN-enabled smart campus network architecture featuring Digital Twin monitoring, predictive analytics, cybersecurity, cloud integration, and unified communication for multi-campus educational institutions.

This project is designed and developed in collaboration with **Vanshika Choube** and created as a part of *ECTE364 - Data Communications* Project Requirements under Computer and Autonomous Systems Engineering course in UOWD.
